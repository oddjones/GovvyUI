Liverpool.Gov.uk Icons

The icon definitions, names and unicode locations for the bulk of the iconset match the standard "font awesome" (v1) syntax.

(note, font awesome's syntax has since been updated, use the syntax given here for liverpool.gov.uk).

In order to place an icon within the page call it using the <i> element using the following notation:

      <i class="iconName iconType{optional}"&nbsp;</i>
where iconName is the name given below and iconType is one of the following:
Icon rendering classes & their impacts
class impact sample
icon-7x font-size:7em (GIGANTO)  
icon-6x font-size:6em (HUGE)  
icon-5x font-size:5em (enormous)  
icon-4x font-size:4em (very large)  
icon-3x font-size:3em (large)  
icon-2x font-size:2em (bigger)  
icon-1-5x font-size:1.5em (emphasized)  
icon-1x font-size:1em (text size) default
